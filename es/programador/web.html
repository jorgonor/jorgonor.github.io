Programador Web

La Web ha evolucionado mucho en los últimos tiempos. Hemos pasado de webs estáticas a aplicaciones Web que emulan una aplicación de escritorio, de Webs con el propósito de devolver un HTML renderizado a infraestructuras con APIs REST y desarrollo en el lado del cliente.

Y un programador Web Frontend, no es más que parte de esa revolución y debe mantenerse actualizado. En mi caso, he desarrollado proyectos utilizando HTML5, CSS3 y Javascript para la parte dinámica, con frameworks como angular.js y Backbone.js

También tengo nociones básicas de maquetado con distintos tipos de plantillas responsive, como materializecss (la plantilla de este portal) o Bootstrap 3. Con este tipo de frameworks de marcado, un desarrollador es capaz de juntar las piezas necesarias para combinar el contenido con los gráficos que quieras destacar en tu página Web, agilizando el proceso de creación y estructura de tus contenidos.

Por último añadir que también puedo realizar Webs como las de toda la vida, me adapto a las necesidades de cada uno.
