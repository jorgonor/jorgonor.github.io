Dentro de este entorno tan competitivo, hace aparición Nativescript. Nativescript es una nueva tecnología open-source de Telerik para poder desarrollar aplicaciones nativas para Android e iOS utilizando como lenguaje de programación Javascript, o también el nuevo primo-hermano de Microsoft, Typescript.
